## 六、负面效果（Debuff）说明

部分道具在提供强力效果的同时会有负面效果，需要玩家权衡：

| Debuff类型 | 说明 | 影响 |
|---|---|---|
| `move_speed_reduction` | 移动速度降低 | 百分比降低（如-10%表示移动速度减少10%） |
| `health_reduction` | 最大生命降低 | 固定值降低（如-2表示最大生命值-2） |
| `attack_speed_reduction` | 攻击速度降低 | 百分比降低（如-8%表示攻击速度减少8%） |
| `crit_chance_reduction` | 暴击率降低 | 百分比降低（如-3%表示暴击率减少3%） |
| `increased_damage_taken` | 受到伤害增加 | 百分比增加（如+10%表示受到伤害增加10%） |
| `damage_reduction` | 伤害减免降低 | 百分比降低（较少使用） |

## 七、道具获得特效

所有道具在获得时都会显示特效反馈：

- **绿色品质**：绿色粒子特效 + 绿色屏幕消息
- **蓝色品质**：蓝色粒子特效 + 蓝色屏幕消息
- **紫色品质**：紫色粒子特效 + 紫色屏幕消息
- **金色品质**：金色粒子特效 + 金色屏幕消息

特效包括：
- 粒子爆炸效果（50个粒子，360度扩散）
- 屏幕消息提示（显示"获得道具"和道具名称）
- 音效反馈（item_pickup音效）
